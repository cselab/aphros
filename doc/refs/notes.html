Geometrical VOF scheme

Two steps:

  • interface reconstruction
  • geometrical flux computation and interface advection

Interface reconstruction

The interface is represented in each cell by a line. Given normal vector, location of the interface and volume fraction are in correspondence.

Conversion is done with routines by [52] Scardovelli2000.
